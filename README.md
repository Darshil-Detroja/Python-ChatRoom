-> Key Components
- User Class
	- Creates and manages users
	- Allows users to join and leave chatrooms
	- Sends messages to the chatroom

-> Message Class
	- Stores message content, sender, and message ID
	- Automatically assigns unique IDs to messages

-> ChatRoom Class
	- Manages users in the chatroom
	- Broadcasts messages to all users
	- Stores and displays chat history

---

-> Features
	- Create multiple users dynamically
	- Join and leave chatroom
	- Send messages in real-time (console-based)
	- Display complete chat history
	- Menu-driven interface for easy interaction
